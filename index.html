Welcome to Decipher, a one-of-a-kind interactive chat experience!

Connect with friends to send scrambled messages, and crack the two-digit code on the bottom of your screen to decipher the hidden messages.

Decipher is a full-stack MERN application built using MongoDB, Express, React, Node, and React Router. Socket.io assisted with real-time chat functionality, and jsonwebtoken and brcrypt Node modules were used to implement token-based authentication.

drinkOn is a happy hour finding application built to encourage users to submit up-to-date information on menus, hours, and drink specials.

Working as part of a three-person team using Git and Github for version control, I was responsible for relational database modeling using PostgreSQL, and implementing CRUD functionality allowing users to upload data, suggest edits, and vote up or down other users’ suggestions.

Built using HTML, CSS, Python, Django, Amazon Web Services, PostgreSQL, and the Google Places API.

Based on the classic carnival game, Whack-A-Mole features three exhilarating levels of mole-whacking fun! Click the moles as they pop up to whack them with your mallet.

Every mole you hit adds one to your score, but be careful, because the moles will burrow back down if you don't whack them in time, and the game speeds up as the timer progresses!

Built using HTML5, CSS3 (including Flexbox and Grid), and jQuery.
